---

## What this project does

### Main features
- View list of cannabis products
- Filter products by:
  - category
  - THC range
  - search by name
- Open product detail page
- Add/remove products from wishlist
- User login/register with JWT auth
- Admin can manage products (create, edit, delete)

---

## Pages

- `/` → Product list page  
- `/product/:id` → Product detail page  
- `/wishlist` → User wishlist  
- `/login` → Login page  
- `/register` → Register page  
- `/admin` → Admin panel (admin only)

## Tech stack

### Frontend
- React (Vite)
- React Router
- Redux Toolkit
- Axios
- Bootstrap

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT auth
- bcrypt

### Deployment
- Frontend on **Vercel**
- Backend on **Render**
- Database on **MongoDB Atlas**
